# 有限要素法の基礎

[0.1]×[0.1]上でポアソン方程式をディリクレ境界条件で解く。
問題設定は、有限要素法概説(新訂版)p.109 例7.5による。

# m=2の場合
下図はメッシュ分割 m=2 の例。

![m=2の三角形メッシュ](figs/mesh_m2.png)

ch7.pyではこのようなメッシュ上でポアソン方程式を
有限要素法により解く。
その結果の中心点でのuの値が0.06249999999999999となり
p.110の表7.3のm=2の場合を再現する。
